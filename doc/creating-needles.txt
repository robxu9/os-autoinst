Some packages are required to run crop.py. So for the first time do:
  zypper in python-imaging python-tk

Example: creating a needle called 'inst-mirror_is_setup'

 1) Run the testcase
    $ os-autoinst/tools/isotovideo PATHTO/openSUSE-NET-i686.iso
 If you want to watch or control the test run:
    $ vncviewer localhost:99
 2) Copy the screenshot you want to use
    $ cp qemuscreenshot/20130101_000000.000000.png os-autoinst/distri/opensuse/needles/inst-mirror_is_setup.png
 3) Use crop.py to create the corresponding needle
    $ os-autoinst/crop.py --tag inst-mirror_is_setup os-autoinst/distri/opensuse/needles/inst-mirror_is_setup.png
    crop.py will show the instructions on your terminal
 4) Profit
